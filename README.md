====================================================
STUDENT TASK MANAGER
====================================================

DESCRIPTION
-----------
A lightweight and easy-to-use task management tool built specifically for students. It helps keep track of assignments, exams, and daily study goals so you never miss a deadline and stay organized throughout the semester.

FEATURES
--------
- Add new tasks with deadlines and subjects.
- View all pending and completed tasks.
- Mark tasks as "Done" once completed.
- Delete or edit existing tasks.
- Simple, distraction-free command-line interface.

PREREQUISITES
-------------
- Python 3.x installed on your system. 
  (You can check if you have it by running `python --version` in your terminal).
- No external libraries required! This project uses standard Python built-in modules.

HOW TO RUN
----------
1. Download or clone the project files to your local machine.
2. Open your terminal or command prompt.
3. Navigate to the folder where you saved the project using the `cd` command.
4. Run the application by typing the following command:
   
   python main.py

HOW TO USE
----------
Launch the application and follow the on-screen prompts in your terminal. You can easily navigate the text menu by typing the corresponding numbers to add your daily subjects, set target dates for assignments, and check off tasks as you finish them.

FUTURE ENHANCEMENTS (Planned)
